## 4. البدء في استخدام CSS
- CSS السطريَّة (Inline CSS)
- CSS الداخليَّة / الدفينة (Embeded CSS)
- CSS الخارجيَّة (External CSS)

## 5. شيفرات CSS
- بنية القاعدة (Rule)
- المُحدِّدات (selectors)
- زوج الخاصية / القيمة (property : value)

## 6. خصائص مظهر النصوص
- المحاذاة `text-align`
- التزيين `text-decoration`
- اتجاه القراءة `direction`
- تحويل حالة الأحرف `text-transformation`
- إزاحة البداية `text-indent`
- اللون `color` (بالاسم، Hexadecimal، rgb)

## 7. خصائص الخطوط Fonts
- `font-family` — العوائل: Serif / Sans-serif / Monospace
- `font-style`
- `font-size`

## 8. خصائص مظهر الروابط
- `a:link`
- `a:visited`
- `a:hover`
- `a:active`

## 9. خصائص الخلفيات Backgrounds
- `background-color`
- `background-image`
- `background-repeat`
- `background-position`
- التحديد بالفئات (classes)
